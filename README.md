# Tic Tac Toe Game

A console-based Tic Tac Toe game developed in C++ that demonstrates fundamental programming concepts such as arrays, loops, functions, and conditional statements.

## Features

- Two-player gameplay (X vs O)
- Dynamic game board display
- Input validation
- Win detection (rows, columns, diagonals)
- Draw detection
- Replay option
- Simple and interactive console interface

## Concepts Used

- Arrays
- Loops
- Functions
- Conditional Statements
- User Input/Output

## How to Compile

Using g++:

```bash
g++ main.cpp -o tictactoe
```

## How to Run

Windows:

```bash
tictactoe.exe
```

Linux/Mac:

```bash
./tictactoe
```
